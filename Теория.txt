1.
Разработчик создаёт: 
1) Класс с определёнными свойствами;
2) Подкласс на его основе, который берёт свойства класса и добавляет свои;
3) Объект подкласса, который также копирует его свойства и добавляет свои. 

Каждый дочерний элемент наследует методы и атрибуты, прописанные в родительском. Он может использовать их все, отбросить часть или добавить новые. При этом заново прописывать эти атрибуты и методы не нужно. 

2.
В отличие от других популярных объектно-ориентированных языков, Java не поддерживает множественое наследование классов.
Но при помощи интерфейсов эту проблему можно решить.

3.
Полиморфизм - понятие из ООП, которое позволяет обрабатывать разные типы данных, принадлежащие разным классам, с помощью одних и тех же методов.

4.
Полиморфизм обеспечивается при помощи механизмов наследования и переопределения методов.

5.
В Java существует два вида полиморфизма:
1)Статический
2)Динамический
